Add SSL

To add a Lets Encrypt SSL certificate to your redirect, follow the instructions below.


Step 1 - Allow Lets Encrypt to validate domain ownership

Before progressing, ensure your domain is pointing to the redirect correctly.

Now open Lets Encrypt in the command line and run the command below:

sudo certbot certonly --email hello@redirectable.co -d redirectable.co --agree-tos --manual

Once you have run the command above, Lets Encrypt will provide you with codes to enter in the boxes below:

{{sslsetupform.as_p}}

Step 2 - Upload the Lets Encrypt SSL Certificate

Once you have clicked confirm above, click enter in the command line to complete the Lets Encrypt process

At this point, Lets Encrypt should have provided you with the required certificates. Use the upload button below to upload the correct certificate file to your redirect then hit "Save".
